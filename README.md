# RAFT Dashboard Implementation

A basic implementation of a dashboard for monitoring and visualizing RAFT consensus algorithm.

## Overview

This project provides a foundational dashboard to visualize the state and behavior of nodes in a RAFT (Reliable, Replicated, Redundant, And Fault-Tolerant) consensus system.

## Features

- Node status monitoring
- Leader/follower/candidate state visualization
- Log replication tracking
- Term and commit index display
